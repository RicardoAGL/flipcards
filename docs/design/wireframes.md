## 1. Overview

This document provides detailed ASCII wireframes for all screens in the Dutch pronunciation learning app. Wireframes are presented in mobile-first format (iPhone) with tablet/desktop variations noted.

**Design Philosophy:** Duolingo-inspired bright, playful aesthetic with clear visual hierarchy and generous touch targets (minimum 44x44px).

## 10. Responsive Breakpoints

### 10.1 Breakpoint Definitions

| Name | Width | Target |
|------|-------|--------|
| xs | 0-374px | iPhone SE |
| sm | 375-413px | iPhone 12/13/14 |
| md | 414-767px | iPhone Plus/Max |
| lg | 768-1023px | iPad Portrait |
| xl | 1024px+ | iPad Landscape, Desktop |

### 10.2 Layout Adaptations Summary

| Component | Mobile (xs-md) | Tablet (lg) | Desktop (xl) |
|-----------|----------------|-------------|--------------|
| Sound tiles | 2-column grid | 4-column grid | 4-column + sidebar |
| Cards | Stacked (80-120-80) | Horizontal (120-180-120) | Horizontal (140-200-140) |
| Quiz options | Full-width stack | 2x2 grid | 2x2 grid |
| Navigation | Bottom tab bar | Side rail | Full sidebar |
| Typography | Base scale | +2px all | +4px all |

## 11. Accessibility Annotations

### 11.1 Focus Order (Tab Navigation)

```
Practice View Focus Order:
1. Back button
2. Progress indicator (read-only)
3. Prefix card (actionable)
4. Center card (actionable)
5. Suffix card (actionable)
6. "Got it" button
7. "Need practice" button
8. "Continue" button
```

### 11.2 Screen Reader Labels

| Element | aria-label |
|---------|------------|
| Prefix card | "Prefix: [letter]. Tap to change" |
| Center card | "Main sound: [sound]. Tap to see pronunciation guide" |
| Suffix card | "Suffix: [letter]. Tap to change" |
| Progress bar | "Progress: [X] of [Y] words completed" |
| Points display | "Total points: [number]" |
| Locked lesson | "Lesson locked. Complete [prerequisite] to unlock" |
