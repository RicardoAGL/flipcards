## 3. Phase 1 Lessons (MVP)

### 3.1 Phase 1 Overview
**Focus:** Same vowel combinations (long vowel sounds)
**Total Lessons:** 8
**Sounds Covered:** aa, ee, oo, uu
**Total Words:** ~92-112 words (depending on exact counts)

### 3.2 Sound Selection Rationale
These four sounds were selected for Phase 1 because:
1. **Phonetic Consistency:** Same letter repeated = predictable pronunciation
2. **Foundational Skills:** Long vowels are common in Dutch
3. **Spanish Contrast:** These sounds differ from Spanish vowel pronunciation
4. **Progressive Difficulty:** Builds confidence before tackling complex combinations

## 7. Content Development Guidelines

### 7.1 Word Selection Process

#### Step 1: Frequency Analysis
- Use Dutch frequency corpus (e.g., SUBTLEX-NL, Corpus Gesproken Nederlands)
- Filter for words containing target sound
- Prioritize top 5000 most common words

#### Step 2: Difficulty Classification
**Beginner Criteria:**
- 1-2 syllables
- Frequency rank < 2000
- Simple consonant patterns (CV, CVC)
- No consonant clusters or max 2-letter clusters

**Advanced Criteria:**
- 2-4 syllables
- Frequency rank 500-5000
- Complex consonant clusters (CCV, CCVC, etc.)
- Compound words acceptable
- Less common vocabulary acceptable

#### Step 3: Pedagogical Review
- Ensure variety in word types (nouns, verbs, adjectives)
- Include culturally relevant vocabulary
- Avoid obscenities or sensitive terms
- Verify pronunciation consistency across Netherlands/Belgium Dutch

#### Step 4: Native Speaker Validation
- Confirm pronunciation accuracy
- Verify naturalness of word selection
- Test with target user (Spanish speaker learning Dutch)

### 7.2 Pronunciation Guide Standards

Each lesson must include:

**IPA Notation:**
- Use standard IPA symbols
- Indicate length with [Àê] where appropriate
- Consistent notation across all lessons

**Spanish Approximation:**
- Use familiar Spanish phonetic references
- Acknowledge when no direct Spanish equivalent exists
- Provide articulation instructions in clear Spanish

**English Approximation:**
- Use common English word examples
- Note differences from English pronunciation
- Provide articulation instructions in clear English

### 7.3 Quiz Distractor Guidelines

**Effective Distractors:**
- Plausible incorrect answers (not obviously wrong)
- Similar sounds from other lessons (tests discrimination)
- Words student has NOT practiced in current lesson
- Mix of familiar and unfamiliar words

**Distractor Sources:**
- Other words with target sound (not in lesson)
- Words with similar but distinct sounds (aa vs. a)
- Words from other lessons (cross-sound confusion)

**Avoid:**
- Distractors with completely different sounds
- Overly similar spellings causing trick questions
- Words with ambiguous pronunciation

## 8. Spaced Repetition Algorithm

### 8.1 Algorithm Specification

**Base Repetition Count:** 3 times per word

**Adaptive Frequency:**
- **Easy Words:** Exactly 3 repetitions, evenly spaced
- **Difficult Words:** 4-6 repetitions, increased frequency
- **Neutral Words:** 3 repetitions, standard spacing

### 8.2 Word Presentation Logic

```
Initial Queue: All words in lesson (random order)

Repetition 1:
  - Show all words once (random order)
  - User marks each as Easy/Difficult/Neutral (or skip)

Repetition 2:
  - Difficult words: higher probability (2x)
  - Neutral/Easy words: standard probability (1x)
  - Random selection from weighted queue

Repetition 3+:
  - Continue until all words seen minimum 3x
  - Difficult words continue appearing until 4-6x threshold met
  - Progress bar shows: "Words remaining: X/Y need more practice"

Quiz Unlock:
  - Triggered when all words >= 3 repetitions
  - Difficult words don't block quiz (but encouraged to practice more)
```

### 8.3 User Marking Interface

**During Practice:**
- After user flips through a word combination, show quick rating:
  - üëç "Got it!" (Easy)
  - ü§î "Need practice" (Difficult)
  - ‚û°Ô∏è "Next" (Neutral - no rating)

**Persistent Marking:**
- User can change rating at any time
- Ratings persist across sessions
- Reset button to clear all ratings for lesson

### 9.2 Quiz Question Selection

**Algorithm:**
1. Pool = all words from current lesson
2. Randomly select 5 unique words
3. For each word, generate 3 distractors
4. Randomize option order (correct answer not always position A)
5. Present questions sequentially

**No Repeats:**
- Same quiz attempt: no word appears twice
- Retakes: randomize new set of 5 words

### 9.3 Scoring Rules

| Event | Points Awarded |
|-------|----------------|
| Correct answer | +20 points |
| Incorrect answer | 0 points (no penalty) |
| Quiz completion | +10 bonus points |
| Perfect score (5/5) | +50 bonus points (total: 160) |

**Passing Score:** 4/5 correct (80%)

**Retake Policy:**
- Unlimited retakes allowed
- Points still awarded on retakes (encourages practice)
- No point decay or penalties
- Each retake generates new random questions
